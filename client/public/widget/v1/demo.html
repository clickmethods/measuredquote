MeasuredQuote — widget embed demo

This is a static reference page showing how a contractor would embed the estimator on their site. The snippet below is the only thing they need to paste; the loader handles iframe sizing and lead callbacks. 

<script
  src="https://embed.measuredquote.com/widget/v1/widget.js"
  data-mq-tenant="ten_demo_ortiz_concrete"
  data-mq-trade="concrete"
  data-mq-language="en"
  data-mq-mount="#mq-estimator"
  async></script>
